## 3. 系统整体架构与模块划分

### 3.1 整体架构图（文字版）

```
┌─────────────────────────────────────────────────────────────────┐
│                    前端壳层 (App.vue)                             │
│     Header (品牌 + 语言选择) | Content | Footer                 │
├──────────────────────────────────────────────────────────────────┤
│                        路由与视图层                               │
│   Landing.vue (首页搜索表单) ↔ Result.vue (结果展示)           │
├──────────────────────────────────────────────────────────────────┤
│                      API 网关 (FastAPI)                          │
│  /api/pois/search  /api/routes/plan  /api/recommendations       │
├──────────────────────────────────────────────────────────────────┤
│                    业务服务层 (Services)                         │
│  POI 查询服务 | 路线规划服务 | 推荐服务 | 日记管理服务           │
├──────────────────────────────────────────────────────────────────┤
│                   核心算法层 (app/core)                          │
│  Graph（图 + Dijkstra） | Trie（前缀查询） | Heap（Top-K）      │
│  Huffman（压缩） | 倒排索引（搜索）                              │
├──────────────────────────────────────────────────────────────────┤
│                   数据持久化层 (app/db)                          │
│  SQLAlchemy ORM → SQLite (POI、设施、用户、日记表)              │
└──────────────────────────────────────────────────────────────────┘
```

## 8. 常见风险与应对
